Тестовое задание:
Создать веб-приложение на Django для управления дисконтными картами.

Список полей:
серия,
номер,
дата выпуска,
дата окончания активности,
дата использования,
сумма,
статус.

Функционал приложения:
1. Список карт с полями: серия, номер, дата окончания активности, статус
2. Поиск по этим же полям
3. Просмотр профиля карты с историей покупок по ней
4. Изменение статуса карты
5. Удаление карты
6. После истечения срока активности карты, у карты проставляется статус "просрочена"

Код разметить на Git-репозитории.

Будет плюсом:
1. Использование Docker
2. Использование Celery
3. Один из методов с использованием DRF
